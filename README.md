# ПСТГУ: Летняя школа, лекция веб-программирования

## Инструкция

```bash
# 1. Склонируйте репозиторий
git clone https://github.com/ddybka/pstgu-web-programming.git

# 2. Скачайте cgi
pip install legacy-cgi

# 3. Запустите скрипт из папки
python main.py
```
